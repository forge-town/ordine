# Contributing

Ordine is not accepting external code contributions yet.

The project is still in the preview stage, and the public APIs, data model, and workflow surface are expected to change before beta. To keep that iteration tight, pull requests from external contributors will be closed until the beta release.

## Current Policy

- External pull requests are not being reviewed or merged yet.
- Maintainer-authored changes and internal collaboration continue as usual.
- A public contribution workflow will be added when the project reaches beta.

## Before Beta

If you are evaluating the project, please treat the repository as preview software and expect breaking changes.
