# Ordine

Meta-orchestration engine for code quality.

Define best practices, compose them into recipes, wire recipes through pipelines, and let AI + scripts enforce them automatically.

## What It Does

- **Best Practices** — Capture coding standards with checklists (Script / LLM checks)
- **Operations** — Atomic coding tasks
- **Recipes** — Operation + Best Practice, bundled as one unit
- **Pipelines** — Chain recipes into multi-step flows
- **Rules** — Policies that trigger pipeline enforcement
- **Skills** — Pluggable AI agent capabilities
- **Projects** — GitHub repo integration with workspace view
- **Jobs** — Track background execution

## Quick Start

```sh
bun install
cd apps/app && bun run db:push
bun dev
```
