# Ordine

元编排引擎，用于代码质量治理。

定义最佳实践，组合为配方，串联成流水线，由 AI + 脚本自动执行。

## 功能

- **最佳实践** — 编码标准，附带检查清单（脚本 / LLM 检查）
- **操作** — 原子级编码任务
- **配方** — 操作 + 最佳实践，打包为一个执行单元
- **流水线** — 将配方串联为多步骤流程
- **规则** — 触发流水线执行的策略
- **技能** — 可插拔的 AI Agent 能力
- **项目** — GitHub 仓库集成与工作区视图
- **任务** — 后台执行跟踪

## 快速开始

```sh
bun install
cd apps/app && bun run db:push
bun dev
```
