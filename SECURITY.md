# Security Policy

Ordine is currently in preview, and there is no public security reporting or triage process yet.

## Current Status

- Public vulnerability submissions are not being accepted before beta.
- There is no published security SLA, advisory process, or private disclosure channel yet.
- The software should be treated as preview software and not relied on for sensitive or production workloads.

## Future Policy

A formal security policy and reporting channel will be published when Ordine reaches beta.
